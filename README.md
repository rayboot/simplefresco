RFresco  -- easy way to use fresco.
========================================


Introduction
--------

```java
RFresco.load(imageUrl)
        .centerInside()
        .circle()
        .into(simpleDraweeView);
```


Use
--------

Gradle:

    com.github.rayboot.simplefresco:fresco-builder:1.2

or

    com.github.rayboot.simplefresco:fresco-pollexor-builder:1.2
